The AI dresses it up.

vibrisse-core is deliberately incomplete. It's a skeleton engineered for Artificial Intelligence.

We banned React Native Gutenberg. We banned page builders. The Core provides the strict boundaries: Vite, Tailwind v4, ACF Pro, and native HTML architecture.

The AI (Cursor, Copilot, or our own Agent) reads the included .ai/AGENTS.md files, understands your project's constraints, and generates compliant business blocks in seconds.

[ The Guardrails ]
  • 1
    Zero Native Gutenberg React. Use exclusively ACF Blocks architecture.
  • 2
    Zero hard-coded colors. Everything routes through theme.json.
  • 3
    Web-First Principle. Native HTML/CSS APIs > External libraries.
